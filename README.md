# Front End Flask REST API
This project implements to [flask-rest-api](https://github.com/piinalpin/flask-rest-api) repository

## Step to run
```
git clone https://github.com/piinalpin/FE-flask-rest-api.git
cd FE-flask-rest-api
npm install
npm run serve
```
